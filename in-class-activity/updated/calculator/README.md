# Simple Calculator App

This is a simple calculator app built with Streamlit.

## How to Run
1. Activate the virtual environment: `venv\Scripts\activate` on Windows, `source venv/bin/activate` on macOS/Linux.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the app: `streamlit run app.py`.
4. Access the app through the provided local URL in your web browser.
